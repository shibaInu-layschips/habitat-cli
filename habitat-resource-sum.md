A brief written summary of the difference between a resource catalog entry, local inventory, a blueprint, and a module.

The difference between a resource catalog entry, local inventory, a blueprint, and a module is that a resource catalog entry is a list of all of your resources and their details, a local inventory is a list of all of your resources and their details on your local machine, a blueprint is a template for creating a module, and a module is a running instance of a blueprint.
